## 🎯 Как использовать (Windows)

### Шаг 1: Запуск
1. Откройте `build/windows/x64/runner/Release/neotun.exe`
2. Приложение автоматически создаст папку `cores` для ядер

### Шаг 2: Установка ядра
1. Перейдите на вкладку **Cores**
2. Нажмите **Download** для нужного ядра (XRay/sing-box/hysteria2)
3. Дождитесь завершения загрузки

### Шаг 3: Добавление конфигурации
**Вариант A: Импорт подписки**
1. Вкладка **Configs** → кнопка **+**
2. Выберите **Import Subscription**
3. Вставьте URL подписки
4. Нажмите **Import**

**Вариант B: Ручное создание**
1. Вкладка **Configs** → кнопка **+**
2. Выберите **Manual Config**
3. Заполните параметры:
   - Name: Имя конфигурации
   - Core: Выберите ядро
   - Protocol: vmess/vless/trojan/ss
   - Address: Адрес сервера
   - Port: Порт
   - UUID/Password: Учетные данные
4. Нажмите **Save**

### Шаг 4: Подключение
1. Выберите конфигурацию из списка
2. Нажмите кнопку **Play** (▶)
3. Дождитесь сообщения "Connected"
4. Проверьте логи - должно быть "✓ System proxy enabled"

### Шаг 5: Проверка
Откройте браузер и перейдите на:
- https://whoer.net/ - проверка IP
- https://www.google.com/ - проверка доступа

## 🔧 Решение проблем

### Проблема: "Connected" но интернет не работает

**Причина:** Системный прокси не настроен автоматически

**Решение 1: Автоматическая настройка**
```
Приложение теперь автоматически настраивает прокси!
Просто переподключитесь:
1. Нажмите Stop
2. Нажмите Play снова
3. Проверьте логи - должно быть "✓ System proxy enabled"
```

**Решение 2: Ручная настройка прокси**
1. Откройте **Настройки Windows** → **Сеть и Интернет** → **Прокси**
2. Включите **Использовать прокси-сервер**
3. Адрес: `127.0.0.1`
4. Порт: `10808` (или другой из конфигурации)
5. Сохраните

**Решение 3: Используйте TUN режим (только sing-box)**
1. Установите Wintun driver: https://www.wintun.net/
2. Выберите конфигурацию с sing-box
3. В настройках выберите **TUN Mode**
4. Подключитесь

### Проблема: Ядро не запускается

**Решение:**
1. Проверьте, что ядро скачано (вкладка Cores)
2. Проверьте логи на ошибки
3. Попробуйте другое ядро
4. Перезапустите приложение

### Проблема: Не могу скачать ядро

**Решение:**
1. Проверьте интернет соединение
2. Попробуйте позже (GitHub может быть недоступен)
3. Скачайте ядро вручную:
   - XRay: https://github.com/XTLS/Xray-core/releases
   - sing-box: https://github.com/SagerNet/sing-box/releases
   - Hysteria2: https://github.com/apernet/hysteria/releases
4. Поместите в папку `cores/`

## 🎨 Режимы работы

### Proxy режим (по умолчанию)
- ✅ Работает со всеми ядрами
- ✅ Не требует дополнительных драйверов
- ✅ Автоматическая настройка системного прокси
- ⚠️ Некоторые приложения могут не использовать прокси

**Рекомендуется для:**
- Веб-браузинга
- Большинства приложений
- Быстрой настройки

### TUN режим (только sing-box на Windows)
- ✅ Полный перехват всего трафика
- ✅ Работает со всеми приложениями
- ⚠️ Требует Wintun driver
- ⚠️ Требует прав администратора

**Рекомендуется для:**
- Игр
- Приложений без поддержки прокси
- Полной изоляции трафика

## 📊 Проверка работы

### Тест 1: Проверка IP
```
1. Откройте https://whoer.net/
2. IP должен измениться на IP вашего сервера
3. DNS должен быть от вашего сервера
```

### Тест 2: Проверка прокси
```powershell
# Windows PowerShell
curl.exe -x socks5://127.0.0.1:10808 https://api.ipify.org
```

### Тест 3: Проверка логов
```
В приложении должны быть сообщения:
✓ Core started
✓ System proxy enabled: 127.0.0.1:10808
✓ Proxy connection successful
```
