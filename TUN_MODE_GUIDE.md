# Руководство по TUN режиму в NeoTUN

## Что такое TUN режим?

**TUN (Network TUNnel)** - это режим работы VPN, который создает виртуальный сетевой интерфейс и перехватывает весь сетевой трафик на уровне операционной системы.

### Отличия от Proxy режима:

| Характеристика | Proxy режим | TUN режим |
|----------------|-------------|-----------|
| Перехват трафика | Только HTTP/HTTPS/SOCKS | Весь TCP/UDP трафик |
| Настройка приложений | Требуется для каждого | Автоматически для всех |
| Производительность | Выше | Немного ниже |
| Совместимость | Не все приложения | Все приложения |
| Права доступа | Не требуются | VPN разрешение (Android) |

## Windows

### Требования

1. **Wintun Driver** - современный TUN драйвер для Windows
   - Скачать: https://www.wintun.net/
   - Или установить автоматически через sing-box

2. **sing-box ядро** - только sing-box поддерживает TUN на Windows
   - XRay и Hysteria2 требуют дополнительной настройки

### Установка Wintun

**Автоматическая установка:**
```bash
# sing-box установит Wintun автоматически при первом запуске TUN
```

**Ручная установка:**
1. Скачайте `wintun.dll` с https://www.wintun.net/
2. Поместите в `C:\Windows\System32\`
3. Или в папку с приложением NeoTUN

### Использование

1. Откройте NeoTUN
2. Выберите конфигурацию с **sing-box** ядром
3. В настройках выберите **TUN Mode**
4. Нажмите **Connect**
5. Приложение автоматически создаст TUN интерфейс

### Проверка работы

```powershell
# Проверить TUN интерфейс
ipconfig /all | findstr "neotun"

# Проверить маршруты
route print

# Тест подключения
curl -x socks5://127.0.0.1:10808 https://api.ipify.org
```

### Отладка

**Проблема: TUN не запускается**
```
Решение:
1. Запустите NeoTUN от имени администратора
2. Проверьте наличие wintun.dll
3. Проверьте логи в приложении
```

**Проблема: Нет интернета в TUN режиме**
```
Решение:
1. Проверьте DNS настройки
2. Убедитесь, что маршрутизация настроена
3. Проверьте firewall Windows
```

## Android

### Требования

- Android 5.0+ (API 21+)
- Разрешение VPN (запрашивается автоматически)

### Использование

1. Откройте NeoTUN
2. Выберите любую конфигурацию
3. В настройках выберите **TUN Mode**
4. Нажмите **Connect**
5. Подтвердите запрос VPN разрешения
6. Приложение создаст VPN подключение

### Особенности Android TUN

- **Полный перехват трафика** - все приложения автоматически используют VPN
- **Split tunneling** - можно исключить приложения из VPN
- **Always-on VPN** - можно включить постоянное VPN подключение
- **Kill switch** - блокировка трафика при отключении VPN

1. Откройте браузер
2. Перейдите на https://whoer.net/
3. Проверьте, что IP изменился
4. Проверьте DNS leak

### Отладка

**Проблема: VPN не подключается**
```
Решение:
1. Проверьте разрешение VPN в настройках Android
2. Отключите другие VPN приложения
3. Перезагрузите устройство
```

**Проблема: Некоторые приложения не работают**
```
Решение:
1. Добавьте приложение в исключения
2. Проверьте настройки маршрутизации
3. Попробуйте другое ядро (sing-box рекомендуется)
```

## Рекомендации

### Когда использовать Proxy режим:

✅ Для веб-браузинга  
✅ Когда нужна максимальная производительность  
✅ Когда не нужны права администратора  
✅ Для тестирования конфигураций  

### Когда использовать TUN режим:

✅ Для игр и приложений  
✅ Когда нужен полный перехват трафика  
✅ Для торрентов и P2P  
✅ Когда приложения не поддерживают прокси  
✅ На Android (рекомендуется)  

---

## Производительность

### Proxy режим:
- Задержка: ~5-10ms
- Пропускная способность: 95-100% от максимальной
- CPU: Низкая нагрузка

### TUN режим:
- Задержка: ~10-20ms
- Пропускная способность: 85-95% от максимальной
- CPU: Средняя нагрузка

---

## Безопасность

### Proxy режим:
⚠️ DNS leak возможен  
⚠️ Некоторые приложения могут обходить прокси  
✅ Не требует системных прав  

### TUN режим:
✅ Полная изоляция трафика  
✅ DNS leak защита  
✅ Kill switch поддержка  
⚠️ Требует VPN разрешение

## FAQ

**Q: Какой режим быстрее?**  
A: Proxy режим немного быстрее, но разница незначительна для большинства задач.

**Q: Можно ли переключаться между режимами?**  
A: Да, просто остановите подключение и выберите другой режим.

**Q: TUN режим работает на всех ядрах?**  
A: На Windows только sing-box. На Android - все ядра.

**Q: Нужны ли права администратора?**  
A: На Windows - да, для TUN режима. На Android - только VPN разрешение.

**Q: Можно ли использовать TUN и Proxy одновременно?**  
A: Нет, нужно выбрать один режим.
