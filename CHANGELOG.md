## Известные проблемы

### Windows
- ⚠️ TUN режим требует Wintun driver
- ⚠️ Требуются права администратора для TUN
- ⚠️ XRay не поддерживает TUN нативно

### Android
- ⚠️ Требуется Android 5.0+ (API 21+)
- ⚠️ Некоторые устройства могут блокировать TUN режим
- ⚠️ Battery optimization может останавливать сервис

### Общие
- ⚠️ GitHub API rate limit при частых обновлениях
- ⚠️ Некоторые ISP могут блокировать протоколы

## Миграция

### С v1.0.0 на v1.1.0

**Автоматическая миграция:**
- Конфигурации сохраняются автоматически
- Ядра остаются на месте
- Настройки переносятся

**Ручные действия:**
1. Для использования TUN режима установите Wintun (Windows)
2. Выберите режим в настройках
3. Переподключитесь

**Обратная совместимость:**
- ✅ Все старые конфигурации работают
- ✅ Proxy режим работает как раньше
- ✅ Никаких breaking changes
